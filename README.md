# 부록. Go에 대해서 몰랐던 부분들 적는 곳

* 프로젝트 루트 디렉토리 기준으로 `mod init` 동작을 통해 `go.mod` 파일을 만들었을 때,  
타 디렉토리의 패키지들은 전체 모듈 이름을 다 적고 나서 마지막에 폴더 이름을 적어야 정상적으로 import 할 수 있음.
* 현재 패키지가 아닌 다른 패키지의 함수는 앞 글자가 대문자인 경우에만 export 됨.
